# ESP32 Smart Sensing System with Blynk.Cloud

A multi-sensor IoT project using ESP32, Ultrasonic, IR, and Sound Sensor with real-time alerts through Blynk.Cloud.

## Features
- Ultrasonic, IR, Sound sensing
- Real-time alerts via Blynk.Cloud
- Optional Supabase logging
- Simple circuit and easy to extend

## Hardware
- ESP32
- HC-SR04 Ultrasonic Sensor
- IR Obstacle Sensor
- Sound Sensor
- Jumper wires, breadboard
